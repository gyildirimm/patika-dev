### Insertion Sort Example

#### [22,27,16,2,18,6] 

##### 1) Yukarıda verilen dizinin sort türüne göre aşamalarını yazınız.

Adım1 : [22,27,16,2,18,6]

Adım2 : [2,27,16,22,18,6]

Adım3 : [2,6,16,22,18,27]

Adım4 : [2,6,16,18,22,27]



##### 2)Big-O gösterimini yazınız.

 O(n^2)

##### 3) Time Complexity: 

###### Average case: O(n^2)

###### Worst case: O(n^2)

###### Best case: O(n)
